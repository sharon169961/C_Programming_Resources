### **Reverse a String Using Pointers**  

#### **Problem Statement:**  
Write a program that reverses a string using pointers.

#### **Input Format:**  
- A single string of up to **100 characters**.

#### **Output Format:**  
- Print the **reversed string**.

#### **Example:**  

##### **Input:**  
```
hello
```
##### **Output:**  
```
olleh
```
